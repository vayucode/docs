---
title: "Exporting customer data from Vayu to CSV"
description: "Export Vayu customer data including names, status, addresses, contracts, contacts, and custom fields to CSV with configurable columns and filters."
---

To export customer information:

1. Navigate to **Customers**
2. Use the **Filters** button to narrow down the customer list if needed
3. Click the **Export / Download** button in the control bar
4. Select the fields you want to include in the export
5. Download the CSV file

### **What's included in customer exports**

Customer exports include:

- Customer name and legal name
- Email and contact information
- Status (Active, Inactive, Churned, Trial)
- Address details (country, state, city, postal code)
- Subsidiary information
- Parent customer relationships
- Number of contracts
- Created date
- Invoice due configuration
- Manual review status
- Any custom fields you've configured

You can customize which columns appear in the export by using the **Settings (gear icon)** to select specific fields before exporting.

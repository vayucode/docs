---
title: "Events"
---

The **Events** page displays raw event data recorded by the system.

Events represent incoming activity before aggregation or billing calculations are applied.

This view is primarily used for validation, investigation, and troubleshooting.

## **Filters**

You can filter events by:

- **Date Range**
- **Events** – Filter by event name or type.
- **Ref** – Search by event reference.
- **Customer Alias** – Locate events associated with a specific customer.

These filters help isolate individual events or investigate activity patterns.
